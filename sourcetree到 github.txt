（1）将 github 的库克隆到本地。（远端） > （选择要克隆的库,点击克隆） > （此时，弹出一个对话框，选择一个本地的路径以储存将要被克隆的github库） >  (然后，选择本地，可以看到刚才选择的文件夹)  >  (把要上传的文件放进这个文件夹里)  > （在 SourceTree打开 这个文件夹） >  (可以在文件状态里看到这个将要被上传的文件)  >  （注意写注释选择提交） >  (之后点击推送。大功告成)




举个例子：
只需要将需要上传的文件放在⁨
shiyi⁩/Documents⁩/SourceTree/computerized-algorithm/
⁩目录下，就可以利用 SourceTree，将文件上传到 github 中的computerized-algorithm库里。

之后，可以将该目录里的文件删除。
